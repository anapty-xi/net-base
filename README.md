📈 Net Base 

Сервис предоставляющий возможность ондовременной манипуляции данными на севрере. Помогает заполнять готовые exel таблицы, экспортировать их, делать бэкапы, просматривать отчеты о работе над данными. Используется внутри отдела крупной российской компании. 

🛠 Технологический стек

Backend: Python, DRF
Auth: JWT
Database: PostgreSQL 15, SQLAlchemy
Frontend: VUE, Bootstrap 5
Tasks: Redis, Celery
Infrastructure: Docker, Docker Compose
Server: nginx

🚀 Ключевые особенности

1. Сервис ускорил работу с таблицами в 3 раза и помог избежать ошибок при их заполнении 
1. SQLAlchemy CORE позволило динамически создавать таблицы из данных пользователя
2. Микросервисная архитектруа, которая упрощает маштабирование
3. Dockerized Secrets: Безопасное управление конфиденциальными данными через Docker Secrets.
4. Clean Architecture для обеспечения легкого маштабирования и последущей разработки
5. Авто тесты позволяют изменять части сервиса и автоматически проверять их
